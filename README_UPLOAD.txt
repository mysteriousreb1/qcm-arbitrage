INSTALLATION GITHUB PAGES

1. Ouvre ton dépôt GitHub : mysteriousreb1/qcm-arbitrage
2. Clique sur "uploading an existing file" ou "Add file" > "Upload files".
3. Glisse-dépose tout le CONTENU du dossier qcm-arbitrage, pas le dossier lui-même.
   Tu dois voir à la racine : index.html, manifest.json, css/, js/, data/, icons/.
4. Clique sur "Commit changes".
5. Va dans Settings > Pages.
6. Source : Deploy from a branch.
7. Branch : main / root.
8. Clique sur Save.
9. Attends 1 à 3 minutes.
10. Ton site sera disponible ici : https://mysteriousreb1.github.io/qcm-arbitrage/

IPHONE

1. Ouvre l'URL dans Safari.
2. Bouton partager.
3. Ajouter à l'écran d'accueil.
